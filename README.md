# CSL 7640: Natural Language Understanding Assignment 2

This repository contains both parts of Assignment 2.

## Folder Structure

- `q1/`: Problem 1, learning Word2Vec embeddings from IIT Jodhpur data
- `q2/`: Problem 2, character-level Indian name generation using recurrent models

## Problem 1

Inside `q1/` you will find:

- dataset collection and preprocessing code
- Word2Vec training code for CBOW and Skip-gram
- semantic analysis and visualization scripts
- final cleaned corpus and selected output files used in the report

## Problem 2

Inside `q2/` you will find:

- the 1000-name training dataset
- character-level implementations of Vanilla RNN, BLSTM, and RNN with attention
- evaluation outputs and generated sample names

## Notes

- Temporary smoke-test results and submission packaging folders are ignored in Git.
- The source files include simple comments written in a student-friendly style.
